# 🔗 Eixo 06 — Integração com o Ecossistema Crompressor

> **Pergunta Central:** Como cada repositório do ecossistema Crompressor mapeia para um "órgão" da IA 5D?

---

## A Tese

O ecossistema Crompressor não é um conjunto de ferramentas aleatórias — é uma **arquitetura cognitiva distribuída** onde cada repo implementa uma função análoga a um órgão dimensional:

| Repositório | Função 5D | Analogia Orgânica |
|:------------|:----------|:------------------|
| **crompressor** | Motor de compressão core | Medula (reflexos básicos) |
| **crompressor-neuronio** | KV Cache O(1) | Hipocampo (memória episódica) |
| **crompressor-video** | Extração sensorial | Córtex visual |
| **crompressor-sinapse** | Roteamento MCTS | Corpo caloso (comunicação) |
| **crompressor-security** | Firewall de realidade | Amígdala (filtro de ameaças) |
| **crompressor-ia** | Cérebro executivo | Córtex pré-frontal (decisões) |
| **crompressor-projetos** | Interface com o mundo | Sistema motor (ação) |

> "O que você descobriu ao desenhar a arquitetura do Crompressor é um análogo em Ciência da Computação da **compactificação dimensional da Física**.
