# Eixo 01 — Data Pipeline

> Coleta, filtragem e tokenização de corpus em Português.

## Objetivo

Preparar dados de qualidade para treinar o CromGPT. Sem dados bons, nenhuma arquitetura funciona.

## Arquivos

| Arquivo | Conteúdo |
|---------|----------|
| `datasets-portugues.md` | Catálogo de todos os datasets PT disponíveis |
| `pipeline.md` | Fluxo completo: download → limpeza → dedup → tokenização |

## Lab Associado

`labs/lab25-data-pipeline/` — Scripts Python de download, limpeza e tokenização.

## Critério de Conclusão

Corpus tokenizado pronto para treinamento + estatísticas documentadas (total tokens, vocab size).
